

-  The Online visualization project is in the VizOnline_Web directory.

-  The java source files in the VizOnline_Web/src/java/ directory  is supposed to be the same as the one in Perspectives/src. If any changes are made to the java source files in Perspectives/src, the modified files should be replace in the VizOnline_Web/src/java/

-  A war file can be created using the VizOnline_Web directory, which can be placed in a tomcat/webapp folder for testing.


-  To be consistent, Changes to the java source files should be made in the Perspectives project and the resulting  files copied to the VizOnline_Web project.
